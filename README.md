Siegmeyer2
==========

A SMACSS Static Boilerplate using Susy and Gulp

Comes with susy grids & typographic baseline out of the box


### Install
Node modules:     
`npm install`      

Ruby gems:         
`bundle`   


Front end dependancies:       
`bower install`        


### Watch      
 
Boot up watch server with gulp:        
`gulp watch`
