Siegmeyer2
==========

A SMACSS Boilerplate using Susy, Sassline, and Gulp

Comes with susy grids & typographic baseline out of the box


### Install
Node modules:     
`npm install`      

Ruby gems:         
`bundle`   


Front end dependancies:       
`bower install`        


### Watch      
 
Boot up watch server with gulp:        
`gulp watch`
